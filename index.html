Rasterization Algorithms

Computer graphics algorithms used for rasterization using JavaScript and Canvas. No graphics libraries were used in this code, only algorithms. Imagine each box in the canvas is a pixel and can be understood as a discrete coordinate. The algorithms graph by painting whole pixels. This means that perfect figures may not be drawn.

You can select the origin position of the system of coordinates. In Upper-left the coordinates increase to the right and to the bottom. In Centered the coordinates increase to the right and to the top.

You can choose the type of graph you would like to paint as well as the algorithm to use. Keep in mind that some algorithms may be better than others. You can also choose the amount of boxes or squares that a row can have in the canvas. A higher number of boxes means can work for better figures and a lower number of boxes can be used to see in detail. For a good performance try to use numbers in the recomended range (10-100).

In algorithms that paint circles, the green pixels represent the pixels that are painted by the original algorithm and the rest are produced by reflection.
