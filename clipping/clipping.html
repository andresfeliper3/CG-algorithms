Clipping algorithms

Computer graphics algorithms used for clipping using JavaScript and Canvas. No graphics libraries were used in this code, only algorithms. The canvas represents the window (entire landscape of what is going to be shown) and the viewport (represents the screen of the device) is graphed according to the algorithm.

You can select the type of graph (line or polygon) and the algorithm used for the clipping. The content of the canvas changes according to the algorithm implementation. Lines only need two points (x1,y1) and (x2,y2) but polygons have vertices so you can choose how many vertices you would like to graph and they are graphed in the order you enter them.

The clipped segments that are inside the viewport are painted in red color. The segments that are rejected and not painted inside the viewport are in black color.
